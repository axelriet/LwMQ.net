About
=====
LwMQ.net is a lightweight and efficient message queuing system facilitating seamless communication between distributed applications by offering a robust and scalable solution.
Features
--------
- **Lightweight**: Minimal resource consumption ensuring high performance.
- **Scalable**: Easily handles increasing loads by distributing messages across multiple nodes.
- **Flexible**: Supports various messaging patterns including publish/subscribe and point-to-point.
- **Easy Integration**: Simple APIs for quick integration with existing systems.
